# Wild Life Habitat Eco Village

A website for the eco-friendly ranch in Hilo, Hawaiʻi — inspiring people worldwide to protect Mother Nature.

## Quick Start

Open `index.html` in your browser, or run a local server:

```bash
npx serve .
```

Then visit `http://localhost:3000`.

## Adding Your Images

Replace the placeholder images by saving your photos with these exact filenames:

### Main Images (`images/`)

| Filename | Description |
|----------|-------------|
| `hero.jpg` | Panoramic ranch or landscape shot (hero banner) |
| `mission.jpg` | Mission section — lush Hawaiian scenery |
| `ranch-overview.jpg` | Wide overview of the ranch |

### Animals (`images/animals/`)

| Filename | Description |
|----------|-------------|
| `cows.jpg` | Cows grazing |
| `sheep.jpg` | Sheep on the hills |
| `chickens.jpg` | Free-range chickens |
| `ducks.jpg` | Ducks by the water |

### Sustainability (`images/sustainability/`)

| Filename | Description |
|----------|-------------|
| `architecture.jpg` | Self-sustained building design overview |
| `rainwater.jpg` | Rainwater collection system |
| `solar.jpg` | Solar panels |
| `wind.jpg` | Wind turbine |

### Airbnb (`images/airbnb/`)

| Filename | Description |
|----------|-------------|
| `exterior.jpg` | Guest house exterior |
| `interior.jpg` | Guest house interior |

### Mau (`images/mau/`)

| Filename | Description |
|----------|-------------|
| `portrait.jpg` | Portrait of Mau |
| `boat.jpg` | The 30-foot self-built boat |

### Gallery (`images/gallery/`)

| Filename | Description |
|----------|-------------|
| `01.jpg` through `06.jpg` | General ranch photos |

**Tips:**
- Use `.jpg` or `.webp` format for best performance
- Recommended size: 1600–2400px wide for hero/large images, 800–1200px for cards
- Keep file sizes under 500 KB when possible

## Deploy

This is a static site — deploy anywhere:

- **GitHub Pages:** Push to a repo and enable Pages
- **Netlify / Vercel:** Drag and drop the folder or connect your repo
- **Any web host:** Upload all files via FTP
